# iw00
iOS assignment 0: Hello World App.

作业 1:

利用SwiftUI或Storyboard完成Hello World App的开发

功能要求如下：

1. 设计按钮1，点击按钮，在App显示Hello world字样。
2. 设计按钮2，通过计数器统计点击按钮2的次数，并在App中显示。

提交要求包括：

1. 上述功能齐全
2. 界面风格不限
3. 用Markdown编写项目文档，并展示运行截屏
